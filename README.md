# Shadowrocket Config

无节点 Shadowrocket 配置仓库。

## 导入链接

正常外出/蜂窝/其他 Wi‑Fi：

```text
https://raw.githubusercontent.com/chb1991/shadowrocket-config/main/Selfconfig.conf
```

家里 Wi‑Fi / 软路由模式：

```text
https://raw.githubusercontent.com/chb1991/shadowrocket-config/main/Selfconfig-HomeWiFi.conf
```

说明：本仓库只保存规则配置，不保存节点、订阅链接、UUID、密码或服务器密钥。
